Overview

How it works

Your Robinhood login stays in Chrome. The helper reads ticker symbols from the selected Robinhood screener, then reads full default-expiration Sell Put quote chains only for stocks that pass the TradingView forecast check.

Step 1

Create your Robinhood screeners

In Robinhood on the web, go to Home and scroll to Lists. Select Create watchlist or screener, choose the stock screener option, add filters with +, review Preview, and select Create.

  1. Log in directly at robinhood.com.
  2. From Home, scroll to Lists and select Create watchlist or screener.
  3. Choose the stock screener option and enter one of the exact names below.
  4. Add the filters that fit your strategy and risk tolerance, review Preview, and select Create.
  5. Repeat until all three screeners exist.

Step 2

Install the Chrome helper

The private-beta helper runs locally in desktop Chrome. It reads list symbols and default-expiration Sell Put quote chains from your logged-in Robinhood tab.

  1. Open the helper download page and download the ZIP.
  2. Extract the ZIP into a folder you can keep.
  3. Open chrome://extensions in Chrome.
  4. Turn on Developer mode, click Load unpacked, and select the extracted folder.
  5. Confirm Chrome shows helper version 0.2.5, then refresh the screener dashboard.

Step 3

Run your first scan

  1. On the dashboard, click Open Robinhood and log in directly on Robinhood.
  2. Keep the Robinhood tab open and return to the dashboard. The helper reuses that tab while it opens screeners and checks option chains.
  3. Select Robinhood as the source.
  4. Select Safe, Standard, or Mini.
  5. Enter your portfolio value and Weekly or Yearly return target, then click Refresh.
Mock mode works without Robinhood. Use it to explore the dashboard before connecting your lists.

Reference

Understand the results

A stock first has to pass TradingView: its minimum analyst target must be above its TradingView current price. Then the helper reads the full visible and scrollable Robinhood Sell Put chain for the default expiration. The server chooses the lowest strike below current price where bid / strike meets the selected weekly return target. The default is 2% weekly, which compounds to about 180.03% yearly over 52 weeks.

SKIP

Changing portfolio value reallocates the existing scan automatically. Changing Weekly or Yearly return reuses the latest TradingView rows and cached Robinhood option quotes. Click Refresh only when you want new Robinhood symbols, fresh TradingView data, and fresh Robinhood put quotes.

Support

Troubleshooting

Helper not detected
Reload or reinstall the extension, then refresh the dashboard.
Slow extraction
The helper must open Robinhood lists and option chains in Chrome. Short lists are faster; larger lists take longer because every TradingView-eligible symbol needs an option-chain read.
Missing symbols
Keep Robinhood open, confirm you are logged in, and verify the exact screener names.
Missing option quotes
Keep the Robinhood tab open and avoid interacting with it during a scan. If a symbol still looks wrong, click Refresh to collect a fresh option-chain snapshot.
Unavailable TradingView row
Retry later or confirm TradingView exposes a forecast page for that symbol.
SKIP status
TradingView passed, but no default-expiration below-current Robinhood put bid reached the selected bid/strike return rule.
Helper update
Download the latest ZIP, extract it, and reload the unpacked extension in Chrome. Extension changes do not take effect until Chrome is reloaded.
