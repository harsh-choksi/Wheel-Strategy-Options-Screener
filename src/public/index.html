First-time setup

Set up CSPs and covered calls

  1. Install the Chrome helper Log in directly on Robinhood and keep that browser tab open.
  2. Set up your strategy Use Manual or Auto strategy sources, save Robinhood screeners, type manual CSP symbols, enter CC rows, or import positions from Robinhood.
  3. Run your scan Choose CSP or CC, select Mock or Robinhood, and click Scan.
