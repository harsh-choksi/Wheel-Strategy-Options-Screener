Private beta install

The helper runs in your Chrome browser, reads ticker symbols from your Robinhood screeners, reads option-chain quote values, and sends only the values needed for screener calculations. Current helper version: 0.2.6.

Chrome does not allow this private helper to install itself automatically. Download the ZIP, load it unpacked, and reload the extension after each helper update.

Download ZIP

Install Steps

Use this same flow whenever you update the helper.

  1. Download the ZIP and extract it to a folder you can keep.
  2. Open chrome://extensions.
  3. Turn on Developer mode.
  4. Click Load unpacked.
  5. Select the extracted helper folder.
  6. Confirm Chrome shows helper version 0.2.6.
  7. Return to the screener and refresh the page.
