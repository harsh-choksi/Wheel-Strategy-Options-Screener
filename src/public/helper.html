Private beta install

The helper runs in your Chrome browser, reads only ticker symbols from your Robinhood lists, and sends those symbols to the screener. Current helper version: 0.1.9.

Download ZIP

Chrome does not allow arbitrary websites to install extensions directly. For a one-click public install, publish this helper in the Chrome Web Store and link to that listing.

Install Steps

Use this flow until the helper is published in the Chrome Web Store.

  1. Download the ZIP and extract it to a folder you can keep.
  2. Open chrome://extensions.
  3. Turn on Developer mode.
  4. Click Load unpacked.
  5. Select the extracted helper folder.
  6. Confirm Chrome shows helper version 0.1.8.
  7. Return to the screener and refresh the page.
